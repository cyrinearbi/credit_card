# Credit Card

## Vue.js credit card 

### Dependencies:
- [Vue.js](https://vuejs.org/)
- [Pug.js](https://pugjs.org/)
- [Stylus](http://stylus-lang.com/)
- [Payment](https://github.com/jessepollak/payment)


## Contribute


``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

```
